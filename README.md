1) Replit account
2) Discord account for developed portal

## Commands
Pre-existing commands are listed below - 
1) Ban (?ban)
2) Kick (?kick)
3) Botinfo (?botinfo)
4) Rules (?rules)
5) Add banned word (?addbannedword)
6) Remove banned word (?removebannedword)
